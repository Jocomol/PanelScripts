# PanelScripts
A few scripts which I use for my xfce4-panel with genmon

## corona
```
usage: corona.py [-h] [-c COUNTRY] [--cases] [--casespermil] [--todaycases] [--deaths]
                 [--todaydeaths] [--recovered] [--todayrecovered] [--active] [--critical]

Reads corona-stats.online and prints out specified numbers.

optional arguments:
  -h, --help            show this help message and exit
  -c COUNTRY, --country COUNTRY
                        select country
  --cases               total cases
  --casespermil         cases per one million
  --todaycases          todays cases
  --deaths              total deaths
  --todaydeaths         todays deaths
  --recovered           total recovered
  --todayrecovered      todays recovered
  --active              active cases
  --critical            critical cases

```
